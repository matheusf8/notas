Este projeto é um sistema de notas desenvolvido com FastAPI (backend) e React + Vite + Material UI (frontend).
Ele permite que usuários se cadastrem, façam login com autenticação JWT e gerenciem suas anotações de forma segura.
🔹 Funcionalidades

    Cadastro de usuários com senha criptografada (bcrypt).

    Login com geração de token JWT.

    Reset de senha.

    CRUD completo de notas (criar, listar, editar e excluir).

    Proteção de rotas: apenas usuários autenticados acessam suas notas.

    Interface moderna com Material UI, incluindo tema claro/escuro.

🔹 Tecnologias usadas

    Backend: FastAPI, SQLAlchemy, JWT, Uvicorn.

    Frontend: React, Vite, Material UI, React Router.

    Banco de dados: SQLite (pode ser trocado por outro).

🔹 Deploy

    Backend preparado para rodar em Render/Railway.

    Frontend preparado para rodar em Vercel/Netlify.

    Configuração de CORS para integração entre frontend e backend.
